# Tables
**Code**

```
| Col 1 | Col 2 | Col 3 |
| :---: | :--- | ---: |
| Row 1, Col 1 | Row 1, Col 2 | Row 1, Col 3 |
| Row 2, Col 1 | Row 2, Col 2 | Row 2, Col 3 |
| Row 3, Col 1 | Row 3, Col 2 | Row 3, Col 3 |

[Note: :---: means center aligned]
[Note: ---: means right aligned]
[Note: :--- means left aligned]
```
**Result**

| Col 1 | Col 2 | Col 3 |
| :---: | :--- | ---: |
| Row 1, Col 1 | Row 1, Col 2 | Row 1, Col 3 |
| Row 2, Col 1 | Row 2, Col 2 | Row 2, Col 3 |
| Row 3, Col 1 | Row 3, Col 2 | Row 3, Col 3 |
